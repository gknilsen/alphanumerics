Denominator of the resampling ratio
cutoff
Type: SystemDouble
Cutoff frequency of the filter, relative to the Nyquist frequency. Set below 1 for better alias suppression and above 1 for less high frequency attenuation. The default cutoff, used by the other constructors, is 1.0.
n
Type: SystemUInt32
See Resampler(UInt32, UInt32, UInt32). The default is 10.
alpha
Type: SystemDouble
Alpha parameter for the Kaiser window. THe default is 5.0.
